Description:

This project accompanies a tutorial available on my site. It implements diffuse bump mapping using ARB OpenGL extensions.


Requirements:

ARB_multitexture
ARB_texture_cube_map
ARB_texture_env_combine
ARB_texture_env_dot3


References:

"Per-Pixel Lighting", Philip Taylor. http://msdn.microsoft.com/library/default.asp?url=/library/en-us/dndrive/html/directx11192001.asp

OpenGL Extension Registry: http://oss.sgi.com/projects/ogl-sample/registry/


Keys:

F1	-	Take a screenshot
Escape	-	Quit

1	-	Draw bump mapped, diffuse colored torus
